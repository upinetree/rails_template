# Rails Application Templates

## Requirement

* Ruby 2.2.3
* Ruby on Rails 4.2.4

## Usage

```
rails new [appname] -BT -d=postgresql -m https://raw.githubusercontent.com/upinetree/rails_template/master/basic.rb
```
